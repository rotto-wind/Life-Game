# Life Game

## start

In cmd:

```cmd
cd .../LifeGame
javac *.java
java init
```

## operation

1. First, you need use mouse to light some cells.
2. Next, you can press the "Enter" button to Start / Pause the process.
3. You can stop the process and light / kill some cells after starting, then start the process again.

## rules

1. Grow: if there are 3 cells around a grid.
2. Die:  if the number of cell around a cell is less than 2 or is more than 3. 

## tip

You can easily revise all these rules and the size of grid.
